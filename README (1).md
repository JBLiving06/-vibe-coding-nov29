## Why Claude Won

All three models produced working tools. But Claude's implementation best matched the original prompt requirement:

> "Build tools that work immediately without requiring the user to have any technical knowledge."

**Claude's advantage:** The artifacts feature. Claude generates shareable HTML files that work with one click. ChatGPT and Gemini produced code that required users to:
- Save files manually
- Navigate file systems
- Potentially configure local servers

For a group of 50+ year-olds exploring AI for the first time, Claude's approach removed all friction.

## Architectural Differences

### AI Career Navigator

**Claude:** Makes runtime API calls to `api.anthropic.com`. Can analyze ANY career with fresh, contextual advice. Requires internet connection.

**ChatGPT:** Embedded ~3,000 lines of domain expertise across 13 industry profiles. Works offline. Limited to pre-defined career categories.

**Gemini:** Embedded database of 50+ specific careers. Works offline. Falls back to generic analysis for unlisted careers.

### Sandwich Generation Calculator

**Claude:** Simulates health decline probabilities. Models care cost inflation. Identifies peak pressure years.

**ChatGPT:** Clean form-based input. 15-year projection. Monthly cost breakdown tables.

**Gemini:** React-based interactive charts. Real-time updates as you adjust inputs. Peak year analysis.

## What is Vibe Coding?

"Vibe coding" is the practice of building software by describing what you want in natural language and letting AI handle the implementation. It's characterized by:

- **Natural language prompts** instead of syntax
- **Iterative refinement** through conversation
- **Focus on outcomes** rather than code structure
- **Rapid prototyping** of functional tools
- **Democratized development** for non-programmers

The term gained traction in 2024-2025 as AI coding assistants became sophisticated enough to build complete applications from descriptions.
